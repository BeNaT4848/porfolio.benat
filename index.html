🚀 Proyectos

BecoMarket - Gestión de Comercio

Java MySQL XAMPP

Beco Market es un proyecto personal desarrollado en Java, con base de datos gestionada en XAMPP (MySQL). Esta aplicación permite organizar y gestionar pedidos, usuarios y productos de manera eficiente. Incluye funcionalidades como registro de usuarios, seguimiento de pedidos y administración de inventario.

El objetivo del proyecto es facilitar el control de las operaciones de un comercio, optimizando procesos y mejorando la experiencia de administración mediante una interfaz intuitiva.
